## üí° Problem-Solving Tips

### **When to Use Which Algorithm:**
- **DFS**: Path finding, cycle detection
- **BFS**: Shortest unweighted paths
- **Dijkstra**: Non-negative weights
- **Floyd-Warshall**: All-pairs shortest paths
